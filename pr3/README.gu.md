# પ્રેક્ટિકલ નંબર 3: SQL DDL કમાન્ડ્સ

આ ડિરેક્ટરીમાં પ્રેક્ટિકલ નંબર 3 નું સોલ્યુશન સામેલ છે.

## ફોલ્ડર વ્યવસ્થાપન
બધા પ્રેક્ટિકલ્સ માટે `dbms-lab` નામનું મુખ્ય ફોલ્ડર બનાવવાની ભલામણ કરવામાં આવે છે. તેની અંદર, દરેક પ્રેક્ટિકલ માટે અલગ સબફોલ્ડર બનાવો, જેમ કે `pr1`, `pr2`, `pr3`, વગેરે.
ઉદાહરણ:
```
dbms-lab/
├── pr1/
├── pr2/
└── pr3/
    ├── create_tables.sql
    ├── insert_data.sql
    ├── ...
```
આ પ્રેક્ટિકલને લગતી તમામ ફાઇલો `pr3` ફોલ્ડરમાં સેવ કરવી જોઈએ.

## ફાઇલો

- `create_tables.sql`: 7 કોષ્ટકો (tables) બનાવવા માટેના DDL કમાન્ડ્સ.

- `insert_data.sql`: કોષ્ટકોમાં ડેટા ઉમેરવા માટેના DDL કમાન્ડ્સ.
- `queries.sql`: ડેટા મેળવવા માટે ડેમો SQL ક્વેરીઝ.
- `pr3.db`: SQLite ડેટાબેઝ ફાઇલ (સ્ક્રીપ્ટ્સ રન કર્યા પછી બને છે).
- `setup_database.py`: ડેટાબેઝ બનાવવા માટેની પાયથોન સ્ક્રીપ્ટ (`create_tables.sql` રન કરે છે).

- `seed_data.py`: ડેટાબેઝમાં માહિતી ભરવા માટેની પાયથોન સ્ક્રીપ્ટ (`insert_data.sql` રન કરે છે).
- `demo_queries.py`: ડેમો ક્વેરીઝ રન કરવા માટેની પાયથોન સ્ક્રીપ્ટ (`queries.sql` રન કરે છે).

## કેવી રીતે રન કરવું

### વિકલ્પ 1: પાયથોન સ્ક્રીપ્ટ્સનો ઉપયોગ (બધી OS માટે ભલામણ કરેલ)
આ સ્ક્રીપ્ટ્સ Windows, Mac અને Linux પર કામ કરે છે. ખાતરી કરો કે તમારી પાસે Python ઇન્સ્ટોલ કરેલું છે.

1.  **ડેટાબેઝ બનાવો:**
    ```bash
    python setup_database.py
    ```
    આ ખાલી કોષ્ટકો સાથે `pr3.db` બનાવશે.

2.  **સેમ્પલ ડેટા ઉમેરો:**
    ```bash
    python seed_data.py
    ```
    આ `pr3.db` માં વિદ્યાર્થીઓ, કોર્સ, યુઝર્સ, પ્રોડક્ટ્સ વગેરે માહિતી ભરશે.

3.  **ડેમો ક્વેરીઝ રન કરો:**
    ```bash
    python demo_queries.py
    ```
    આ ડેટા અને સંબંધો બતાવવા માટે સેમ્પલ ક્વેરીઝ એક્ઝિક્યુટ કરશે.

### વિકલ્પ 2: "DB Browser for SQLite" નો ઉપયોગ (GUI)

1.  **સ્ટ્રક્ચર ઇમ્પોર્ટ કરો:**
    - DB Browser ઓપન કરો.
    - **મહત્વપૂર્ણ:** જો કોઈ અન્ય ડેટાબેઝ ઓપન હોય, તો પહેલા **Close Database** પર ક્લિક કરો.
    - **File** -> **Import** -> **Database from SQL file...** પર ક્લિક કરો.
    - `create_tables.sql` સિલેક્ટ કરો. `pr3.db` તરીકે સેવ કરો.

2.  **ડેટા ઇમ્પોર્ટ કરો:**
    - `pr3.db` ઓપન હોય ત્યારે, **Execute SQL** ટેબ પર જાઓ.
    - **Open SQL file** (ફોલ્ડર આઇકોન) પર ક્લિક કરો અથવા `insert_data.sql` માંથી કન્ટેન્ટ કોપી-પેસ્ટ કરો.
    - **Execute all** (પ્લે બટન) પર ક્લિક કરો.
    - સેવ કરવા માટે **Write Changes** પર ક્લિક કરો.

3.  **ક્વેરીઝ રન કરો:**
    - **Execute SQL** ટેબ પર જાઓ.
    - `queries.sql` ઓપન કરો.
    - પરિણામ જોવા માટે એક્ઝિક્યુટ કરો.

### વિકલ્પ 3: કમાન્ડ લાઇનનો ઉપયોગ મેન્યુઅલી

```bash
# ડેટાબેઝ બનાવવા માટે
sqlite3 pr3.db < create_tables.sql


# ડેટા ઉમેરવા માટે
sqlite3 pr3.db < insert_data.sql

# ક્વેરીઝ રન કરવા માટે
sqlite3 pr3.db < queries.sql
```
